# Price Comparison Frontend

A modern React + Tailwind CSS frontend for comparing product prices across Amazon, Flipkart, and Myntra.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will open at `http://localhost:3000`

## Build

```bash
npm run build
```
